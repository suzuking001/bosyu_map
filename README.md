# 浜松市　こども園/保育園 募集・申込状況マップ

浜松市の公開オープンデータを利用し、募集人数と申込人数を地図上に表示する静的Webアプリです。複数の施設種別データを統合し、年齢・施設種別でフィルタできます。

GitHub Pages: https://suzuking001.github.io/bosyu_map/

## 特徴
- 浜松市オープンデータから施設CSVを取得して表示
- 年齢・施設種別でフィルタ
- 募集状況に応じたマーカー色分け（空きあり=緑 / 空き1=黄 / 空きなし・募集なし=赤 / 情報なし=白）
- 施設名と募集・申込数のラベルを常時表示
- 現在地表示と2km/5km圏のガイド円
- About / Licenses モーダルでライセンス情報を表示

## 使い方
1. `index.html` をブラウザで開きます。
2. メニューから年齢・施設種別を選択します。
3. 地図上のマーカーをクリックすると詳細が表示されます。
